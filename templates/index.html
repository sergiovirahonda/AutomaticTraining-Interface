COVID-19 PREDICTOR

A Convolutional Neural Network capable of predicting whether a patient's chest X-ray image belongs to a COVID-19 positive case, Viral Pneumonia or a normal patient.

ABOUT THE PROJECT

This project is based on the COVID-19 Radiography dataset available at https://www.kaggle.com/tawsifurrahman/covid19-radiography-database - such recopilation has been created by a team of researchers from Qatar, Bangladesh and some collaborators from Pakistan and Malaysia. Essentially it contains several chest X-ray images for COVID-19 positive cases along with Normal and Viral Pneumonia images. Behind this interface there's an API that instantiate a Convolutional Neural Network trained automatically with a DevOps approach.
